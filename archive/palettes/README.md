# Palette experiments (archived)

Load on pages that include `palette-warm.js`:

- **Earth** (now production default): `?palette=earth` → `theme-palette-earth.css`
- **Social / crimson**: `?palette=social` (aliases: `warm`, `crimson`, `apricot`) → `theme-palette-warm.css`
- **Red room**: `?palette=red` — removes experimental sheets

Production root pages no longer load this script; use **`../../assets/theme-site.css`** instead.
